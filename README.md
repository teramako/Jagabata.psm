# Jagabata.psm 🥔🧈
<img src="docs/img/Jagabata.png" width="256" height="256" align="left"/>

PowerShell module to operate AWX/AnsibleTower using [Rest API].

- Required: PowerShell 7
- Supports Windows, GNU/Linux, macOS
